# grida

## Commands

```sh
$ grida init
$ grida login
$ grida logout
$ grida config
$ grida build
$ grida deploy
$ grida ui pull
$ grida ui push
$ grida ui eject
$ grida eject
$ grida studio
$ grida docs
$ grida doctor
$ grida generate
$ grida help
$ grida run
$ grida run <widget> # or grida preview <widget>
$ grida . # open current directoy in Grida
$ grida publish # publish grida package
```
